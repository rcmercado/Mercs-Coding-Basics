    CSS aka Cascading Style Sheets.

    CSS is embeded inside HTML. In order to start stylizing the webpage you must add a style tag just below of the title tag.

    Lets change the color of headings in here.

    You must add a CSS Style rule which tells the browser which parts of the webpage is CSS and Declaration. Declartions tells the browser how to style that part.

    First add a style tag just below the title tag.


    To style all of the H2 tags and change their color use the curly braces, { }

    .

    Example

  • h2 { }

    • Put the Declaration inside which have properties and values.

    Example

  • h2 { color: green}

    • To change the color of multiple header sizes (h2) in the same line just added with a comma.

    Eample

  • h2,h3,h4{Color:green;}

    • It is possible to change the color of any aspect of the text style. such as the strong tag, emphasis tag and bold tag. Sometimes common colors are just, too common like red, green and blue. To use a wider array of colors use the RGB property followed by parenthesis.

    Eample

  • h2 {color: rgb (214,60,212);}.

    • Go to a website such as Colorpicker.com select a color and use the R,G,B number values and type them inside the parenthesis in the RGB property. Remember to use commas.

    To change the background of the webpage use the body selector.

    Example

  • body {background-color:rgb(); or color:;}
    • Now lets change the color of a specific thing like one paragraph.

    Lets do with this very paragraph you're currently reading. In order to do change the color we have to give the paragraph an ID. The ID tells CSS which paragraph to modify.

    The ID should be placed inside the opening paragraph tag, just after the P with a space. Type a very unique name inside the quotation marks. It needs to be something that will never appear in the code.

    Example

  • id=""

    • Type the selector in the style tag. Use a hash tag #. The Hash tells CSS that whatever is typed next will be an ID. After adding the hash and the name of the ID add the curly braces and properties with the values

    ID's are case sensitive. Be sure to type it exactly after the Hash tag.

    Adding ID's can be done to any HTML tag using the same step above.

    Classes can be used for assigning a particular element to a group. To add a class atribute just do what was done for the ID's, but do not use a hash tag in the style tag. Instead, use period(.), do not use spaces in this class tag, it won't work.

    Example

  • Class=""

    • Instead of creating unique ID's for everything you'd like modified just use a Class. You can use the same class name for everything you want to be colored the same. You can copy and paste the class tag with the name to anything you want to.
